# Submission package note

## What this directory is

This tree is the **reviewer-facing submission package** for the supplementary code and data accompanying *External Falsifiable Measurement for Long-Chain Reasoning*. It was **derived by selective export** from a larger internal archival reconstruction workspace (`ExternalFalsifiableMeasurementforLongChain`). **Only materials judged necessary for review** (paper alignment, runnable entrypoints, disclosures, minimal data/results) are retained here.

## What was removed relative to the archive

Internal packaging logs, phase-by-phase audit Markdown, upstream inventory snapshots, and other **non-reviewer** artifacts were **not** copied. Full reconstruction provenance remains in the archival bundle on the author side, not in this zip.

## What was not modified externally

- The LaTeX source project and the read-only upstream code tree were **not** altered by this export.
