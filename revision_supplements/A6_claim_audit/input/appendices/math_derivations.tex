\section{Mathematical Derivations}\label{app:math}

\subsection{Ordinal Verification-Entropy Formula}

The verification-entropy is defined as:
\begin{equation}
H_v = -\sum_{i=1}^{k} \frac{f_i}{N} \log_2 \frac{f_i}{N}
\end{equation}

This is an ordinal index, not an absolute entropy measure. The values should be interpreted relative to the protocol-defined categories, not as cardinal quantities.

\subsection{Model Selection Criteria}

We use AIC and BIC for model comparison within the ordinal framework. Lower values indicate better fit. The held-out $R^2$ is reported as a diagnostic indicator, not as variance explained.
